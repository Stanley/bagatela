Here you will find everything from use case scenarios to detailed API documentation, all writen to help you seamlessly integrate your application with public transport data and extra services. If you have any problems or requests please submit new issue and I'll response as quickly as I can.

Raw data Get access to stops and timetables which are gathered form various sources, but are all presented in a unified form. Graph API As opposed to raw data, graph API will give you access to analyzed and fully connected data with handful of services build on top of it.

This website as well as the back-end are open source. Please help me by forking the project and adding to it.
